# KJV Harmony Companion

**The complete King James Version Bible study suite for Macedonia Primitive Baptist Church and believers everywhere.**

Live website: https://sametheredge85-crypto.github.io/kjv-companion/

## What is this?
A free, easy-to-use collection of tools to help you study, preach, and enjoy the KJV Bible.  
Everything works offline after the first visit (PWA — you can even “Add to Home Screen” on your phone like a real app).

## Main Features
- Bible Harmonizer (see how verses connect)
- Annex Library & Doctrinal Themes
- Preacher Suite & Sermon Builder
- Audio Library (sermons & recordings)
- Image & Visual Library
- First Principles Engine
- Bible Questions & Context Restorer
- All powered by `bibleData.js`

## How to Use
1. Visit the site above
2. Tap the menu or go straight to any tool (harmonizer.html, preacher-suite.html, etc.)
3. Install it on your phone for offline use!
